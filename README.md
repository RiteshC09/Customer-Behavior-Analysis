# Customer Behavior Analysis
End-to-end customer shopping behavior analysis using Python, SQL and Power BI to uncover spending patterns, customer segmentation and business insights.
Project Overview

This project analyzes 3,900 customer transactions across multiple product categories to identify:
- Customer spending patterns
- High-value customer segments
- Product performance and ratings
- Impact of discounts and subscriptions
- Revenue contribution by demographic groups

The workflow includes:
- Data cleaning and transformation using Python
- Business analysis using SQL in PostgreSQL
- Interactive dashboard creation in Power BI

📊 Dataset Information
Total Records: 3,900
Total Columns: 18

Key Features:
- Customer Demographics: Age, Gender, Location, Subscription Status
- Purchase Details: Item, Category, Purchase Amount, Season, Size, Color
- Behavioral Metrics: Discount Applied, Purchase Frequency, Review Rating, Shipping Type
- Customer Activity: Previous Purchases, Promo Code Usage

Data Cleaning & Processing (Python)
Performed using Pandas, NumPy, and PostgreSQL integration

Key steps:
- Imported and explored dataset using pandas
- Handled missing values using median imputation
- Standardized column names to snake_case

Created new features:
- age_group
- purchase_frequency_days
- Removed redundant columns
- Loaded cleaned dataset into PostgreSQL database
